Detailed Description

A program to solve the coin problem (https://en.wikipedia.org/wiki/Coin_problem)

The coin problem is a math riddle also known as the Frobenius coin problem, postage stamp problem, and Chicken McNugget Theorem. The problem asks that given 2 items with different positive weights/value (for example coins/stamps with monetary value), what is the largest total weight/value that cannot be achieved using any combination of those two items? If the two items (a & b) are relatively prime, this can be solved using the formula (a*b)-(a+b). If they are not relatively prime, there is no solution.
